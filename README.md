# Serendipity Casa Vacanze Torino

Sito web per una casa-vacanze.

## Caratteristiche Principali

-   **Design Responsivo:** Layout fluido che si adatta a desktop, tablet e mobile.
-   **Multi-lingua:** Supporto nativo per Italiano (IT) e Inglese (EN) con switch istantaneo.
-   **Calendario Disponibilit√†:** Integrazione in tempo reale con Google Sheets (CSV) per mostrare le date libere/occupate.
-   **Galleria Fotografica:** Lightbox personalizzata con navigazione touch, zoom e supporto tastiera.
-   **Mappa Interattiva:** Integrazione Google Maps per la posizione e i punti di interesse.
-   **Prenotazione Diretta:** Sistema "mail-to" precompilato per inviare richieste dirette all'host.

## Tecnologie

-   **HTML5:** Struttura semantica.
-   **CSS3:** Styling personalizzato (senza framework pesanti), Flexbox/Grid, variabili CSS per il tema.
-   **JavaScript (Vanilla):** Logica per il calendario, cambio lingua, galleria e menu mobile.

## Struttura

-   `index.html`: Home page (unica pagina principale).
-   `explore.html`: Pagina dettagli sulla zona/quartiere.
-   `style.css`: Foglio di stile principale.
-   `IMAGES/`: Cartella contenente gli asset grafici ottimizzati (.webp).

## Setup Locale

Per visualizzare il sito in locale:

1.  Clonare la repository.
2.  Aprire `index.html` nel browser o avviare un server locale (es. `python3 -m http.server`).
